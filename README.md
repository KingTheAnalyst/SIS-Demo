# SIS / SIP HTML CSS Investor Demo

This is a static front-end demonstration for the Sports Intelligence System (SIS) and Sports Intelligence Platform (SIP).

## What this updated version includes

- A more professional investor-ready landing page
- Clear SIS vs SIP positioning
- Natural-language question and response section
- Plain-English statistics explanations
- Sample match dashboard
- Sample intelligence report
- Feature module representation
- Telegram and WhatsApp delivery previews
- Future API output preview
- Investor update dashboard
- Roadmap section
- Responsive design for desktop and mobile

## How to open in VS Code

1. Open this folder in VS Code.
2. Right-click `index.html`.
3. Select `Open with Live Server`.
4. If Live Server is not installed, install it from VS Code Extensions.

## Important investor note

This is a visual prototype using sample data only. It is not a production backend system and it does not provide guaranteed predictions.
